# RTK Movie App
# Clone this repository
```bash
$ git clone https://github.com/mubasharali-dev/RTK-App.git
```

# Go into the repository
```bash
$ cd rtk-app
$ npm install
$ npm run dev
```
# For build
```bash
$ npm run build
```

# For updating
```bash
$ npm update
```
